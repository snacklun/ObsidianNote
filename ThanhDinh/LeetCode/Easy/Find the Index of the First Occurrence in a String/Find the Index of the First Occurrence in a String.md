# Description

Given two strings `haystack` and `needle`, return the index of the **first occurrence** of `needle` in `haystack`, or `-1` if `needle` is not part of `haystack`.

---

## Examples

**Example 1**

Input: haystack = "sadbutsad", needle = "sad" Output: 0 Explanation: "sad" occurs at indices 0 and 6, but the first occurrence is at index 0.

## Constraints

- 1 <= haystack.length, needle.length <= 10^4
- Both `haystack` and `needle` consist only of lowercase English letters.
